# Ordena
Ordena is a Windows Forms desktop planner built with VB.NET and MySQL. It helps users organize events using a monthly calendar view where they can add, edit, and delete schedules. Events are stored in a database, displayed with color highlights, and filtered by category for easy tracking.

It Features the following:
- Monthly calendar display
- Add new events with title, date, category, description, and color
- Edit or delete events by clicking the date
- Upcoming events panel with category filter
- Data is stored in MySQL (XAMPP)

How to Run:
- Open XAMPP and start Apache and MySQL.
- Import the ordena database and events table.
- Open the project in Visual Studio.
- Make sure MySql.Data is installed.
- Run the program.

How to Use:
- Click a date to add an event.
- Click a date with an event to edit or delete it.
- Use the category dropdown to filter upcoming events.
